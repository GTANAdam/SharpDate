## Setup

To get Kair

##### Go CLI
```sh
go get github.com/GuilhermeCaruso/Kair
```
##### Go DEP
```sh
dep ensure -add github.com/GuilhermeCaruso/Kair
```
##### Govendor
```sh
govendor fetch github.com/GuilhermeCaruso/Kair
```
